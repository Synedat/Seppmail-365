# Seppmail-365

> Reference repository for self-hosted SEPPmail plus Exchange Online integration.

**Partner resource by Synedat for the SEPPmail ecosystem.**

## Why this repository exists

Repository wrapper for the SEPPmail365 PowerShell module documentation and practical integration pointers for self-hosted or partner-operated appliances.

This repository is structured for public consumption and easy discovery across topics such as SEPPmail, Exchange Online, Microsoft 365, Azure, API automation, PowerShell and operational runbooks.

## Included content

- Documentation-first repository wrapper and entry point

## Quick start

1. Use this repo as an easy entry point to the official SEPPmail365 documentation.
2. Verify PowerShell and Exchange Online module prerequisites first.
3. Use tenant-specific reports before creating or changing connectors and rules.
